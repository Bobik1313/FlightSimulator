A small project to integrate JSBSim into Unity.

CONTROLS

- LEFT STICK  - Pitch(↕) / Roll(↔)
- RIGHT STICK - Rudder (↔)
- RIGHT TRIGGER - Throttle Up
- LEFT TRIGGER  - Throttle Down

FEATURES

- JSBSim integration
- Basic aircraft controls
- Simple flight physics
- Unity visualization
_______________________________________
- Unity 2022.3+
- Windows x64
